# Dados agregados

Este diretório contém apenas dados agregados mensais usados na análise.

Arquivo principal
- `monthly_aggregates.csv`

Colunas
- `month`: mês de referência (YYYY-MM)
- `volume_total`: total de migrações concluídas no mês
- `sucesso_72h`: migrações concluídas em ≤ 72 horas no mês

Cobertura
- julho/2020 a junho/2025 (60 meses)

Confidencialidade
- Não há microdados, identificadores de clientes ou credenciais.
